
# proxmox_virtual_environment_ha_resources (Data Source)

Retrieves the list of Proxmox VE High Availability resources.




## Schema

### Read-Only

- `groups` (List of String) HA group names.
- `id` (String) The ID of this resource.
- `sids` (List of String) HA resource SIDs.
- `states` (List of String) Desired states of the HA resources.
