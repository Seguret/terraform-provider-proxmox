# proxmox_virtual_environment_time (Resource)

Manages the timezone configuration of a Proxmox VE node.

## Example Usage

### UTC Timezone

```terraform
resource "proxmox_virtual_environment_time" "pve_utc" {
  node_name = "pve"
  timezone  = "UTC"
}
```

### Europe Timezone

```terraform
resource "proxmox_virtual_environment_time" "pve_berlin" {
  node_name = "pve"
  timezone  = "Europe/Berlin"
}
```

### US Timezone

```terraform
resource "proxmox_virtual_environment_time" "pve_est" {
  node_name = "pve"
  timezone  = "America/New_York"
}
```

### Multiple Nodes

```terraform
resource "proxmox_virtual_environment_time" "pve1" {
  node_name = "pve1"
  timezone  = "Europe/London"
}

resource "proxmox_virtual_environment_time" "pve2" {

}

```
