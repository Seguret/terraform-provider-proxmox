# proxmox_virtual_environment_node_disk_directory (Resource)

Manages a directory-based disk storage on a Proxmox VE node.

## Example Usage

### Basic Directory Storage

```terraform
resource "proxmox_virtual_environment_node_disk_directory" "storage" {
  node_name = "pve"
  device    = "/dev/sdb"
  name      = "custom-storage"
  add_storage = true
}
```

### Directory with ext4 Filesystem

```terraform
resource "proxmox_virtual_environment_node_disk_directory" "ext4_storage" {
  node_name  = "pve"
  device     = "/dev/sdc"
  name       = "vm-storage"
  filesystem = "ext4"
  add_storage = true
}
```

### Directory with XFS Filesystem

```terraform
resource "proxmox_virtual_environment_node_disk_directory" "xfs_storage" {
  node_name  = "pve"
  device     = "/dev/sdd"
  name       = "backup-storage"
  filesystem = "xfs"
  add_storage = true
}
```

### Multiple Directory Storages

```terraform
resource "proxmox_virtual_environment_node_disk_directory" "vm_storage" {
  node_name  = "pve"
  device     = "/dev/sdb"
  name       = "vm-data"
  filesystem = "ext4"
  add_storage = true
}

resource "proxmox_virtual_environment_node_disk_directory" "backup_storage" {
  node_name  = "pve"
  device     = "/dev/sdc"
  name       = "backups"
  filesystem = "xfs"
  add_storage = true
}
```
