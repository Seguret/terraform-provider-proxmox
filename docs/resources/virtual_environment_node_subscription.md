# proxmox_virtual_environment_node_subscription (Resource)

Manages the subscription key for a Proxmox VE node.

## Example Usage

### Add Subscription Key

```terraform
variable "proxmox_subscription_key" {
  type      = string
  sensitive = true
}

resource "proxmox_virtual_environment_node_subscription" "pve" {
  node_name = "pve"
  key       = var.proxmox_subscription_key
}
```

### Multiple Nodes with Subscription

```terraform
variable "proxmox_subscription_key" {
  type      = string
  sensitive = true
}

resource "proxmox_virtual_environment_node_subscription" "pve1" {
  node_name = "pve1"
  key       = var.proxmox_subscription_key
}

resource "proxmox_virtual_environment_node_subscription" "pve2" {
  node_name = "pve2"
  key       = var.proxmox_subscription_key
}

resource "proxmox_virtual_environment_node_subscription" "pve3" {
  node_name = "pve3"
  key       = var.proxmox_subscription_key
}
```
