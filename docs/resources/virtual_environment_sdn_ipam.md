# proxmox_virtual_environment_sdn_ipam (Resource)

Manages a Proxmox VE SDN IPAM provider.

## Example Usage

### NetBox IPAM

```terraform
resource "proxmox_virtual_environment_sdn_ipam" "netbox" {
  ipam  = "netbox"
  type  = "netbox"
  url   = "https://netbox.example.com/api"
  token = var.netbox_api_token
}
```

### phpIPAM Provider

```terraform
resource "proxmox_virtual_environment_sdn_ipam" "phpipam" {
  ipam    = "phpipam-prod"
  type    = "phpipam"
  url     = "https://phpipam.example.com/api"
  token   = var.phpipam_token
  section = 1
}
```

### Proxmox VE IPAM

```terraform
resource "proxmox_virtual_environment_sdn_ipam" "pve" {
  ipam = "pve-ipam"
  type = "pve"
}
```
