# proxmox_virtual_environment_sdn_vnet (Resource)

Manages a Proxmox VE SDN VNet.

## Example Usage

### Basic VNet

```terraform
resource "proxmox_virtual_environment_sdn_vnet" "vnet1" {
  vnet    = "vnet1"
  zone    = "zone-simple"
  comment = "Basic virtual network"
}
```

### VLAN-Tagged VNet

```terraform
resource "proxmox_virtual_environment_sdn_vnet" "vlan_100" {
  vnet         = "vlan100"
  zone         = "zone-vlan"
  tag          = 100
  comment      = "VLAN 100 virtual network"
  vlan_aware   = true
}
```

### Multiple VNets

```terraform
resource "proxmox_virtual_environment_sdn_vnet" "production" {
  vnet    = "prod"
  zone    = "zone-vxlan"
  comment = "Production network"
}

resource "proxmox_virtual_environment_sdn_vnet" "staging" {
  vnet    = "stage"
  zone    = "zone-vxlan"
  comment = "Staging network"
}

resource "proxmox_virtual_environment_sdn_vnet" "testing" {
  vnet    = "test"
  zone    = "zone-vxlan"
  comment = "Testing network"
}
```

### VLAN-Aware VNet with Alias

```terraform
resource "proxmox_virtual_environment_sdn_vnet" "guest_vnet" {
  vnet         = "guest"
  zone         = "zone-vlan"
  alias        = "Guest Network"
  tag          = 200
  vlan_aware   = true
  comment      = "Guest access virtual network"
}
```
