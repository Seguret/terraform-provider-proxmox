# proxmox_virtual_environment_sdn_zone (Resource)

Manages a Proxmox VE SDN zone.

## Example Usage

### Simple SDN Zone

```terraform
resource "proxmox_virtual_environment_sdn_zone" "simple" {
  zone    = "zone-simple"
  type    = "simple"
  comment = "Simple SDN zone for basic networking"
}
```

### VLAN Zone

```terraform
resource "proxmox_virtual_environment_sdn_zone" "vlan_zone" {
  zone    = "zone-vlan"
  type    = "vlan"
  bridge  = "vmbr0"
  comment = "VLAN-based SDN zone"
}
```

### VXLAN Zone

```terraform
resource "proxmox_virtual_environment_sdn_zone" "vxlan_zone" {
  zone    = "zone-vxlan"
  type    = "vxlan"
  comment = "VXLAN overlay network"
  mtu     = 1450
}
```

### EVPN Zone with Controller and IPAM

```terraform
resource "proxmox_virtual_environment_sdn_zone" "evpn_zone" {
  zone       = "zone-evpn"
  type       = "evpn"
  controller = "evpn-controller"
  ipam       = "pve-ipam"
  dns        = "pve-dns"
  dns_zone   = "internal.example.com"
  comment    = "EVPN zone with full integration"
}
```

## Schema

### Required

- `type` (String) Zone type ('simple', 'vlan', 'qinq', 'vxlan', 'evpn').
- `zone` (String) The SDN zone name (identifier).

### Optional

- `bridge` (String) The bridge interface (for vlan/qinq zones).
- `comment` (String) A description for this zone.
- `controller` (String) EVPN controller name (for evpn zones).
- `dns` (String) DNS plugin name.
- `dns_zone` (String) DNS domain.
- `ipam` (String) IPAM plugin name.
- `mtu` (Number) MTU value for the zone.
- `peers` (String) Comma-separated list of VXLAN peer addresses (for vxlan zones).
- `reverse_dns` (String) Reverse DNS plugin name.
- `tag` (Number) VLAN tag (for vlan/qinq zones).
- `vrf_vxlan` (Number) VRF VXLAN tag (for evpn zones).

### Read-Only

- `id` (String) The ID of this resource.
