# proxmox_virtual_environment_notification_endpoint_webhook (Resource)

Manages a Proxmox VE webhook notification endpoint.

## Example Usage

### Basic Webhook

```terraform
resource "proxmox_virtual_environment_notification_endpoint_webhook" "basic" {
  name = "webhook-basic"
  url  = "https://example.com/webhook"
}
```

### Webhook with POST Method

```terraform
resource "proxmox_virtual_environment_notification_endpoint_webhook" "post" {
  name    = "webhook-post"
  url     = "https://api.example.com/alerts"
  method  = "POST"
  comment = "POST webhook for alerts"
}
```

### Multiple Webhooks

```terraform
resource "proxmox_virtual_environment_notification_endpoint_webhook" "slack" {
  name   = "slack-webhook"
  url    = "https://hooks.slack.com/services/YOUR/WEBHOOK/URL"
  method = "POST"
}

resource "proxmox_virtual_environment_notification_endpoint_webhook" "custom" {
  name    = "custom-webhook"
  url     = "https://internal.example.com/notify"
  method  = "PUT"
  comment = "Custom internal webhook"
}
```
