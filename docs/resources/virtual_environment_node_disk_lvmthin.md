# proxmox_virtual_environment_node_disk_lvmthin (Resource)

Manages an LVM-thin pool on a Proxmox VE node disk.

## Example Usage

### Basic LVM-Thin Pool

```terraform
resource "proxmox_virtual_environment_node_disk_lvmthin" "thin_pool" {
  node_name     = "pve"
  device        = "/dev/sdb"
  name          = "thin-pool"
  volume_group  = "vg_vms"
  add_storage   = true
}
```

### LVM-Thin Pool without Auto Storage

```terraform
resource "proxmox_virtual_environment_node_disk_lvmthin" "thin_manual" {
  node_name     = "pve"
  device        = "/dev/sdc"
  name          = "tp-data"
  volume_group  = "vg_data"
  add_storage   = false
}
```

### Multiple LVM-Thin Pools

```terraform
resource "proxmox_virtual_environment_node_disk_lvmthin" "thin_vms" {
  node_name     = "pve"
  device        = "/dev/sdb"
  name          = "thin-vms"
  volume_group  = "vg_fast"
  add_storage   = true
}

resource "proxmox_virtual_environment_node_disk_lvmthin" "thin_backup" {
  node_name     = "pve"
  device        = "/dev/sdc"
  name          = "thin-backup"
  volume_group  = "vg_backup"
  add_storage   = true
}
```
