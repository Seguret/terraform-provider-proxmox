# proxmox_virtual_environment_hosts (Resource)

Manages the /etc/hosts file of a Proxmox VE node.

## Example Usage

### Basic Hosts Configuration

```terraform
resource "proxmox_virtual_environment_hosts" "pve" {
  node_name = "pve"
  entries   = <<-EOT
    127.0.0.1 localhost.localdomain localhost
    ::1 localhost.localdomain localhost
    192.168.1.100 pve.example.com pve
  EOT
}
```

### Complete Cluster Hosts File

```terraform
resource "proxmox_virtual_environment_hosts" "cluster" {
  node_name = "pve1"
  entries   = <<-EOT
    127.0.0.1 localhost.localdomain localhost
    ::1 localhost.localdomain localhost
    
    # Cluster nodes
    192.168.1.100 pve1.example.com pve1
    192.168.1.101 pve2.example.com pve2
    192.168.1.102 pve3.example.com pve3
    
    # Services
    192.168.1.50 backup.example.com backup
    192.168.1.51 dns.example.com dns
    192.168.1.52 ntp.example.com ntp
  EOT
}
```

### Hosts with External Services

```terraform
resource "proxmox_virtual_environment_hosts" "with_external" {
  node_name = "pve"
  entries   = <<-EOT
    127.0.0.1 localhost.localdomain localhost
    ::1 localhost.localdomain localhost
    
    # Local node
    192.168.1.100 pve.example.com pve
    
    # External services
    203.0.113.10 mail.example.com mail
    203.0.113.11 ldap.example.com ldap
    203.0.113.12 nfs.example.com nfs
  EOT
}
```
