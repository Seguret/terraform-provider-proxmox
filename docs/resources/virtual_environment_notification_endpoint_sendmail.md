# proxmox_virtual_environment_notification_endpoint_sendmail (Resource)

Manages a Proxmox VE sendmail notification endpoint.

## Example Usage

### Basic Sendmail Endpoint

```terraform
resource "proxmox_virtual_environment_notification_endpoint_sendmail" "local" {
  name = "sendmail-local"
}
```

### Sendmail with Recipients

```terraform
resource "proxmox_virtual_environment_notification_endpoint_sendmail" "alerts" {
  name          = "sendmail-alerts"
  from_address  = "proxmox@example.com"
  mailto        = ["ops@example.com", "admin@example.com"]
  comment       = "System alerts via sendmail"
}
```

### Sendmail with User Recipients

```terraform
resource "proxmox_virtual_environment_notification_endpoint_sendmail" "users" {
  name          = "sendmail-users"
  from_address  = "alerts@example.com"
  author        = "Proxmox System"
  mailto_user   = ["admin@pve", "operator@pve"]
  comment       = "Notify Proxmox users"
}
```

### Multiple Sendmail Endpoints

```terraform
resource "proxmox_virtual_environment_notification_endpoint_sendmail" "ops" {
  name         = "sendmail-ops"
  from_address = "ops-alerts@example.com"
  mailto       = ["ops-team@example.com"]
  author       = "Proxmox Alerts"
}

resource "proxmox_virtual_environment_notification_endpoint_sendmail" "security" {
  name         = "sendmail-security"
  from_address = "security-alerts@example.com"
  mailto       = ["security@example.com"]
  author       = "Proxmox Security"
}
```
