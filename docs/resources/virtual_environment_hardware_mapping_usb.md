# proxmox_virtual_environment_hardware_mapping_usb (Resource)

Manages a Proxmox VE cluster-level USB hardware mapping.

## Example Usage

### USB Device Mapping

```terraform
resource "proxmox_virtual_environment_hardware_mapping_usb" "usb_device" {
  name    = "usb-dongle"
  comment = "USB security dongle"
  map = [
    "node=pve,id=1234:5678"
  ]
}
```

### Multiple USB Devices

```terraform
resource "proxmox_virtual_environment_hardware_mapping_usb" "usb_devices" {
  name    = "usb-peripherals"
  comment = "USB devices for VM"
  map = [
    "node=pve,id=1234:5678,path=1-1",
    "node=pve,id=abcd:ef00,path=1-2"
  ]
}
```

### USB Hub Mapping

```terraform
resource "proxmox_virtual_environment_hardware_mapping_usb" "usb_hub" {
  name    = "usb-hub"
  comment = "USB hub with attached devices"
  map = [
    "node=pve1,id=0451:2077"
  ]
}
```
