# proxmox_virtual_environment_apt_repository (Resource)

Manages a standard Proxmox VE APT repository on a node.

## Example Usage

### Enable Enterprise Repository

```terraform
resource "proxmox_virtual_environment_apt_repository" "enterprise" {
  node_name = "pve"
  handle    = "pve-enterprise"
  enabled   = true
}
```

### Use No-Subscription Repository

```terraform
resource "proxmox_virtual_environment_apt_repository" "no_subscription" {
  node_name = "pve"
  handle    = "pve-no-subscription"
  enabled   = true
}
```

### Manage Multiple Repositories

```terraform
resource "proxmox_virtual_environment_apt_repository" "no_sub" {
  node_name = "pve"
  handle    = "pve-no-subscription"
  enabled   = true
}

resource "proxmox_virtual_environment_apt_repository" "test_release" {
  node_name = "pve"
  handle    = "pve-test-release"
  enabled   = false  # Disabled by default
}

resource "proxmox_virtual_environment_apt_repository" "ceph_reef" {
  node_name = "pve"
  handle    = "ceph-reef-no-subscription"
  enabled   = true
}
```

### Disable Repository

```terraform
resource "proxmox_virtual_environment_apt_repository" "disable_experimental" {
  node_name = "pve"
  handle    = "pve-test-release"
  enabled   = false
}
```
