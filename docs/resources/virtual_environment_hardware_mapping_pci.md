# proxmox_virtual_environment_hardware_mapping_pci (Resource)

Manages a Proxmox VE cluster-level PCI hardware mapping.

## Example Usage

### GPU Mapping

```terraform
resource "proxmox_virtual_environment_hardware_mapping_pci" "nvidia_gpu" {
  name    = "nvidia-gpu"
  comment = "NVIDIA GPU mapping"
  map = [
    "node=pve1,path=0000:81:00.0,id=10de:2684,iommu-group=0"
  ]
}
```

### Multiple GPUs Across Nodes

```terraform
resource "proxmox_virtual_environment_hardware_mapping_pci" "gpus" {
  name    = "cluster-gpus"
  comment = "GPU mapping across cluster"
  map = [
    "node=pve1,path=0000:81:00.0,id=10de:2684",
    "node=pve2,path=0000:82:00.0,id=10de:2684",
    "node=pve3,path=0000:83:00.0,id=10de:2684"
  ]
}
```

### Network Adapter Mapping

```terraform
resource "proxmox_virtual_environment_hardware_mapping_pci" "nic" {
  name    = "network-adapter"
  comment = "Network card passthrough"
  map = [
    "node=pve,path=0000:04:00.0,id=8086:1521"
  ]
}
```
