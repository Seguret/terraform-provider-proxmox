# proxmox_virtual_environment_sdn_subnet (Resource)

Manages a Proxmox VE SDN subnet within a VNet.

## Example Usage

### Basic Subnet

```terraform
resource "proxmox_virtual_environment_sdn_subnet" "subnet1" {
  vnet   = "vnet1"
  subnet = "10.0.0.0/24"
}
```

### Subnet with Gateway

```terraform
resource "proxmox_virtual_environment_sdn_subnet" "subnet_gw" {
  vnet    = "vnet1"
  subnet  = "10.0.1.0/24"
  gateway = "10.0.1.1"
}
```

### Subnet with DHCP and DNS

```terraform
resource "proxmox_virtual_environment_sdn_subnet" "dhcp_subnet" {
  vnet             = "vnet1"
  subnet           = "10.0.2.0/24"
  gateway          = "10.0.2.1"
  dhcp_dns_server  = "10.0.1.10"
  dns_zone_prefix  = "local"
}
```

### Multiple Subnets in VNet

```terraform
resource "proxmox_virtual_environment_sdn_subnet" "prod_subnet" {
  vnet    = "prod-vnet"
  subnet  = "10.1.0.0/24"
  gateway = "10.1.0.1"
  snat    = true
}

resource "proxmox_virtual_environment_sdn_subnet" "dev_subnet" {
  vnet    = "dev-vnet"
  subnet  = "10.2.0.0/24"
  gateway = "10.2.0.1"
  snat    = false
}
```
