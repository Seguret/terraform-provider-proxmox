# proxmox_virtual_environment_notification_endpoint_gotify (Resource)

Manages a Proxmox VE Gotify notification endpoint.

## Example Usage

### Basic Gotify Endpoint

```terraform
resource "proxmox_virtual_environment_notification_endpoint_gotify" "alerts" {
  name   = "gotify-alerts"
  server = "https://gotify.example.com"
}
```

### Gotify with Token

```terraform
resource "proxmox_virtual_environment_notification_endpoint_gotify" "secure" {
  name    = "gotify-secure"
  server  = "https://gotify.example.com"
  token   = var.gotify_token
  comment = "Gotify endpoint with authentication"
}
```

### Multiple Gotify Endpoints

```terraform
resource "proxmox_virtual_environment_notification_endpoint_gotify" "internal" {
  name    = "gotify-internal"
  server  = "https://internal-gotify.corp.local"
  token   = var.internal_gotify_token
  comment = "Internal Gotify server"
}

resource "proxmox_virtual_environment_notification_endpoint_gotify" "cloud" {
  name    = "gotify-cloud"
  server  = "https://gotify.cloud.example.com"
  token   = var.cloud_gotify_token
  comment = "Cloud-hosted Gotify"
}
```
