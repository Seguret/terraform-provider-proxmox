# proxmox_virtual_environment_node_service (Resource)

Manages the desired run state of a system service on a Proxmox VE node.

## Example Usage

### Start Service

```terraform
resource "proxmox_virtual_environment_node_service" "pveproxy" {
  node_name = "pve"
  service   = "pveproxy"
  state     = "started"
}
```

### Restart Service

```terraform
resource "proxmox_virtual_environment_node_service" "pvedaemon" {
  node_name = "pve"
  service   = "pvedaemon"
  state     = "restarted"
}
```

### Multiple Services

```terraform
resource "proxmox_virtual_environment_node_service" "pveproxy" {
  node_name = "pve"
  service   = "pveproxy"
  state     = "started"
}

  node_name = "pve"

  node_name = "pve"

```

### Stop Service

```terraform
resource "proxmox_virtual_environment_node_service" "stopped" {
  node_name = "pve"
  service   = "ceph-mon"
```
