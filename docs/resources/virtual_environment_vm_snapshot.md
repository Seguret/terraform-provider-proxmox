# proxmox_virtual_environment_vm_snapshot (Resource)

Manages a Proxmox VE virtual machine snapshot.

## Example Usage

### Simple VM Snapshot

```terraform
resource "proxmox_virtual_environment_vm_snapshot" "backup" {
  vmid      = 100
  node_name = "pve"
  snap_name = "backup"
}
```

### VM Snapshot with Description

```terraform
resource "proxmox_virtual_environment_vm_snapshot" "before_update" {
  vmid        = 100
  node_name   = "pve"
  snap_name   = "before-kernel-update"
  description = "Snapshot taken before kernel update on 2024-01-15"
}
```

### VM Snapshot with Memory State

```terraform
resource "proxmox_virtual_environment_vm_snapshot" "live_snapshot" {
  vmid      = 100
  node_name = "pve"
  snap_name = "live-state"
  vm_state  = true  # Include memory state
  description = "Live snapshot with memory state"
}
```

### Multiple Snapshots

```terraform
resource "proxmox_virtual_environment_vm_snapshot" "daily" {
  vmid      = 100
  node_name = "pve"
  snap_name = "daily-$(date +%Y%m%d)"
  description = "Daily backup snapshot"
}

resource "proxmox_virtual_environment_vm_snapshot" "before_patch" {
  vmid      = 100
  node_name = "pve"
  snap_name = "before-patch"
  vm_state  = true
  description = "Pre-patch snapshot with memory"
}
```
