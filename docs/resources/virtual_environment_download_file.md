# proxmox_virtual_environment_download_file (Resource)

Downloads a file from a URL to a Proxmox VE node storage.

## Example Usage

### Download ISO Image

```terraform
resource "proxmox_virtual_environment_download_file" "ubuntu_iso" {
  node_name    = "pve"
  storage      = "local"
  content_type = "iso"
  file_name    = "ubuntu-22.04-live-server-amd64.iso"
  url          = "https://releases.ubuntu.com/22.04/ubuntu-22.04-live-server-amd64.iso"
}
```

### Download Container Template

```terraform
resource "proxmox_virtual_environment_download_file" "ubuntu_template" {
  node_name    = "pve"
  storage      = "local"
  content_type = "vztmpl"
  file_name    = "ubuntu-22.04-standard_22.04-1_amd64.tar.zst"
  url          = "http://download.proxmox.com/images/system-ct/ubuntu-22.04-standard_22.04-1_amd64.tar.zst"
}
```

### Download ISO with Checksum Verification

```terraform
resource "proxmox_virtual_environment_download_file" "verified_iso" {
  node_name             = "pve"
  storage               = "local"
  content_type          = "iso"
  file_name             = "debian-11-genericcloud-amd64.img"
  url                   = "https://cloud.debian.org/images/cloud/bullseye/latest/debian-11-genericcloud-amd64.img"
  checksum              = "1234567890abcdef"
  checksum_algorithm    = "sha256"
  verify_tls            = true
}
```

### Download Multiple Files

```terraform
resource "proxmox_virtual_environment_download_file" "iso_ubuntu" {
  node_name    = "pve"
  storage      = "local"
  content_type = "iso"
  file_name    = "ubuntu-22.04.iso"
  url          = "https://releases.ubuntu.com/22.04/ubuntu-22.04-live-server-amd64.iso"
}

resource "proxmox_virtual_environment_download_file" "iso_debian" {
  node_name    = "pve"
  storage      = "local"
  content_type = "iso"
  file_name    = "debian-11.iso"
  url          = "https://cdimage.debian.org/debian-cd/current/amd64/iso-dvd/debian-11.6.0-amd64-dvd-1.iso"
}
```

## Schema

### Required

- `content_type` (String) The content type: 'iso' or 'vztmpl'.
- `file_name` (String) The target filename on the storage.
- `node_name` (String) The name of the Proxmox VE node.
- `storage` (String) The storage name to download to.
- `url` (String) The source URL to download from.

### Optional

- `checksum` (String) Expected checksum for verification.
- `checksum_algorithm` (String) Checksum algorithm (md5, sha1, sha256, etc.).
- `verify_tls` (Boolean) Whether to verify the TLS certificate of the download URL.

### Read-Only

- `file_id` (String) The full volume ID of the file in storage.
- `id` (String) The ID of this resource.
- `size` (Number) The size of the downloaded file in bytes.
