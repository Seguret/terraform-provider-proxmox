# proxmox_virtual_environment_acme_plugin (Resource)

Manages a Proxmox VE ACME DNS challenge plugin.

## Example Usage

### Cloudflare DNS Challenge Plugin

```terraform
variable "cloudflare_token" {
  type      = string
  sensitive = true
}

resource "proxmox_virtual_environment_acme_plugin" "cloudflare" {
  plugin_id = "cloudflare"
  type      = "dns"
  api       = "cf"
  data      = "CF_Token=${var.cloudflare_token}"
}
```

### Route53 DNS Challenge Plugin

```terraform
variable "aws_access_key" {
  type      = string
  sensitive = true
}

variable "aws_secret_key" {
  type      = string
  sensitive = true
}

resource "proxmox_virtual_environment_acme_plugin" "route53" {
  plugin_id = "aws-route53"
  type      = "dns"
  api       = "aws"
  data      = <<-EOT
    AWS_ACCESS_KEY_ID=${var.aws_access_key}
    AWS_SECRET_ACCESS_KEY=${var.aws_secret_key}
    AWS_DEFAULT_REGION=us-east-1
  EOT
}
```

### Standalone HTTP Challenge

```terraform
resource "proxmox_virtual_environment_acme_plugin" "standalone" {
  plugin_id = "standalone"
  type      = "standalone"
}
```

### DNS Plugin with Node Restriction

```terraform
resource "proxmox_virtual_environment_acme_plugin" "dns_nodes" {
  plugin_id         = "cloudflare-prod"
  type              = "dns"
  api               = "cf"
  data              = "CF_Token=your-token-here"
  nodes             = "pve1,pve2"
  validation_delay  = 120
}
```

### Required

### Optional

- `nodes` (String) Comma-separated list of nodes to restrict the plugin to.
