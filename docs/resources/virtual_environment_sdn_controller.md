# proxmox_virtual_environment_sdn_controller (Resource)

Manages a Proxmox VE SDN controller.

## Example Usage

### Simple Controller

```terraform
resource "proxmox_virtual_environment_sdn_controller" "simple" {
  controller = "simple-controller"
  type       = "simple"
}
```

### EVPN Controller

```terraform
resource "proxmox_virtual_environment_sdn_controller" "evpn" {
  controller = "evpn-controller"
  type       = "evpn"
  asn        = 65000
  ebgp       = false
}
```

### BGP Controller

```terraform
resource "proxmox_virtual_environment_sdn_controller" "bgp" {
  controller = "bgp-controller"
  type       = "bgp"
  asn        = 65001
  peers      = "192.168.1.1,192.168.1.2"
}
```

### ISIS Controller

```terraform
resource "proxmox_virtual_environment_sdn_controller" "isis" {
  controller = "isis-controller"
  type       = "isis"
  node       = "pve1"
}
```

## Schema

### Required

- `controller` (String) The SDN controller name (identifier).
- `type` (String) The SDN controller type (evpn, bgp, isis, or simple).

### Optional

- `asn` (Number) The BGP/EVPN autonomous system number.
- `ebgp` (Boolean) Enable eBGP (inter-AS routing). Only applicable for EVPN controllers.
- `node` (String) The cluster node name (for ISIS controllers).
- `peers` (String) Comma-separated list of BGP peer addresses.

### Read-Only

- `id` (String) The ID of this resource.
