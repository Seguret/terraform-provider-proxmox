# proxmox_virtual_environment_sdn_dns (Resource)

Manages a Proxmox VE SDN DNS provider.

## Example Usage

### PowerDNS Provider

```terraform
resource "proxmox_virtual_environment_sdn_dns" "powerdns" {
  dns  = "powerdns"
  type = "powerdns"
  url  = "http://powerdns.example.com:8001"
  key  = var.powerdns_api_key
  ttl  = 3600
}
```

### PowerDNS with Default TTL

```terraform
resource "proxmox_virtual_environment_sdn_dns" "powerdns_default" {
  dns  = "pdns-main"
  type = "powerdns"
  url  = "https://dns.internal.example.com:8443"
  key  = var.powerdns_key
  ttl  = 300
}
```
