# proxmox_virtual_environment_certificate (Resource)

Manages a custom TLS certificate on a Proxmox VE node.

## Example Usage

### Upload Custom Certificate

```terraform
resource "proxmox_virtual_environment_certificate" "custom" {
  node_name   = "pve"
  certificate = file("${path.module}/certs/proxmox.crt")
  private_key = file("${path.module}/certs/proxmox.key")
  restart     = true
}
```

### Let's Encrypt Certificate (via external provider)

```terraform
resource "proxmox_virtual_environment_certificate" "letsencrypt" {
  node_name   = "pve"
  certificate = file("${path.module}/letsencrypt/fullchain.pem")
  private_key = file("${path.module}/letsencrypt/privkey.pem")
  restart     = true
  force       = true  # Override existing certificate
}
```

### Self-Signed Certificate

```terraform
resource "proxmox_virtual_environment_certificate" "selfsigned" {
  node_name   = "pve"
  certificate = file("${path.module}/selfsigned/cert.pem")
  private_key = file("${path.module}/selfsigned/key.pem")
  restart     = false  # Can restart manually later
}
```

### Certificate Update Without Restart

```terraform
resource "proxmox_virtual_environment_certificate" "update" {
  node_name   = "pve"
  certificate = file("${path.module}/certs/new-cert.pem")
  private_key = file("${path.module}/certs/new-key.pem")
  restart     = false
}
```
