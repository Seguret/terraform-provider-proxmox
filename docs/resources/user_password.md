# proxmox_user_password (Resource)

Manages a Proxmox VE user password. This resource allows changing user passwords via the API.

## Example Usage

### Set User Password

```terraform
variable "user_password" {
  type      = string
  sensitive = true
}

resource "proxmox_user_password" "admin" {
  user_id  = "admin@pve"
  password = var.user_password
}
```

### Update Root Password

```terraform
resource "proxmox_user_password" "root" {
  user_id  = "root@pam"
  password = var.root_password
}
```

### Set Password for Multiple Users

```terraform
variable "user_passwords" {
  type      = map(string)
  sensitive = true
}

resource "proxmox_user_password" "users" {
  for_each = var.user_passwords
  
  user_id  = each.key
  password = each.value
}
```
