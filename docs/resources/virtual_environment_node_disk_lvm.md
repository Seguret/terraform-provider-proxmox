# proxmox_virtual_environment_node_disk_lvm (Resource)

Manages an LVM volume group on a Proxmox VE node disk.

## Example Usage

### Basic LVM Volume Group

```terraform
resource "proxmox_virtual_environment_node_disk_lvm" "lvm_vms" {
  node_name   = "pve"
  name        = "vg_vms"
  device      = "/dev/sdb"
  add_storage = true
}
```

### LVM Volume Group Without Auto Storage

```terraform
resource "proxmox_virtual_environment_node_disk_lvm" "lvm_data" {
  node_name   = "pve"
  name        = "vg_data"
  device      = "/dev/sdc"
  add_storage = false
}
```

### Multiple LVM Volume Groups

```terraform
resource "proxmox_virtual_environment_node_disk_lvm" "lvm_vms" {
  node_name   = "pve"
  name        = "vg_vms"
  device      = "/dev/sdb"
  add_storage = true
}

resource "proxmox_virtual_environment_node_disk_lvm" "lvm_backup" {
  node_name   = "pve"
  name        = "vg_backup"
  device      = "/dev/sdc"
  add_storage = true
}

resource "proxmox_virtual_environment_node_disk_lvm" "lvm_data" {
  node_name   = "pve"
  name        = "vg_data"
  device      = "/dev/sdd"
  add_storage = false
}
```
