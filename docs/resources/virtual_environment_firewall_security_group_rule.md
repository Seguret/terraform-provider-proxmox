# proxmox_virtual_environment_firewall_security_group_rule (Resource)

Manages a rule inside a Proxmox VE cluster firewall security group.

## Example Usage

### Allow HTTP and HTTPS

```terraform
resource "proxmox_virtual_environment_firewall_security_group_rule" "allow_http" {
  security_group = "web-servers"
  type           = "in"
  action         = "ACCEPT"
  protocol       = "tcp"
  dport          = "80"
  comment        = "Allow HTTP"
}

resource "proxmox_virtual_environment_firewall_security_group_rule" "allow_https" {
  security_group = "web-servers"
  type           = "in"
  action         = "ACCEPT"
  protocol       = "tcp"
  dport          = "443"
  comment        = "Allow HTTPS"
}
```

### Database Access Rules
