# proxmox_virtual_environment_notification_filter (Resource)

Manages a Proxmox VE notification filter.

## Example Usage

### Basic Filter

```terraform
resource "proxmox_virtual_environment_notification_filter" "all" {
  name = "all-notifications"
}
```

### Severity-Based Filter

```terraform
resource "proxmox_virtual_environment_notification_filter" "important" {
  name         = "important-only"
  min_severity = "warning"
  max_severity = "error"
  comment      = "Filter for important notifications only"
}
```

### Rule-Based Filter

```terraform
resource "proxmox_virtual_environment_notification_filter" "rules_example" {
  name    = "custom-rules"
  comment = "Custom filter rules"
  rules   = ["type=backup", "severity=error"]
}
```

### Disabled Filter

```terraform
resource "proxmox_virtual_environment_notification_filter" "test" {
  name    = "test-filter"
  disable = true
  comment = "Test filter - currently disabled"
}
```
