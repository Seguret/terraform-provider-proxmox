# proxmox_virtual_environment_container_snapshot (Resource)

Manages a Proxmox VE LXC container snapshot.

## Example Usage

### Simple Container Snapshot

```terraform
resource "proxmox_virtual_environment_container_snapshot" "backup" {
  vmid      = 200
  node_name = "pve"
  snap_name = "backup"
}
```

### Container Snapshot with Description

```terraform
resource "proxmox_virtual_environment_container_snapshot" "before_config" {
  vmid        = 200
  node_name   = "pve"
  snap_name   = "before-config-change"
  description = "Snapshot before major configuration update"
}
```

### Multiple Container Snapshots

```terraform
resource "proxmox_virtual_environment_container_snapshot" "hourly" {
  vmid        = 200
  node_name   = "pve"
  snap_name   = "hourly-backup"
  description = "Hourly backup snapshot"
}

resource "proxmox_virtual_environment_container_snapshot" "before_upgrade" {
  vmid        = 200

  description = "Snapshot before application upgrade"

```

### Snapshots for Multiple Containers
