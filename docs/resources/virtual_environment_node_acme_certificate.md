# proxmox_virtual_environment_node_acme_certificate (Resource)

Orders or renews an ACME-issued TLS certificate for a Proxmox VE node. The node must have an ACME account and domain configuration applied beforehand.

## Example Usage

### Order ACME Certificate

```terraform
resource "proxmox_virtual_environment_node_acme_certificate" "pve" {
  node_name = "pve"
}
```

### Force Certificate Renewal

```terraform
resource "proxmox_virtual_environment_node_acme_certificate" "pve_force" {
  node_name = "pve"
  force     = true
}
```

### Certificate with Dependencies

```terraform
resource "proxmox_virtual_environment_acme_account" "letsencrypt" {
  name    = "default"
  contact = "admin@example.com"
}

resource "proxmox_virtual_environment_node_acme_certificate" "pve" {
  node_name = "pve"
  
  depends_on = [
    proxmox_virtual_environment_acme_account.letsencrypt
  ]
}
```
