# proxmox_virtual_environment_dns (Resource)

Manages the DNS configuration of a Proxmox VE node.

## Example Usage

### Google DNS

```terraform
resource "proxmox_virtual_environment_dns" "google_dns" {
  node_name = "pve"
  search    = "example.com"
  dns1      = "8.8.8.8"
  dns2      = "8.8.4.4"
}
```

### Cloudflare DNS

```terraform
resource "proxmox_virtual_environment_dns" "cloudflare_dns" {
  node_name = "pve"
  search    = "example.com"
  dns1      = "1.1.1.1"
  dns2      = "1.0.0.1"
}
```

### Corporate DNS with Multiple Servers

```terraform
resource "proxmox_virtual_environment_dns" "corporate" {
  node_name = "pve"
  search    = "corp.example.com"
  dns1      = "10.0.1.10"     # Primary DNS
  dns2      = "10.0.1.11"     # Secondary DNS
  dns3      = "8.8.8.8"       # Public fallback
}
```

### Multiple Nodes with Same DNS

```terraform
resource "proxmox_virtual_environment_dns" "pve1" {
  node_name = "pve1"
  search    = "datacenter.local"
  dns1      = "10.0.1.10"
  dns2      = "10.0.1.11"
}

resource "proxmox_virtual_environment_dns" "pve2" {
  node_name = "pve2"
  search    = "datacenter.local"
  dns1      = "10.0.1.10"
  dns2      = "10.0.1.11"
}

resource "proxmox_virtual_environment_dns" "pve3" {
  node_name = "pve3"
  search    = "datacenter.local"
  dns1      = "10.0.1.10"
  dns2      = "10.0.1.11"
}
```
