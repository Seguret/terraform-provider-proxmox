# proxmox_virtual_environment_notification_matcher (Resource)

Manages a Proxmox VE notification matcher.

## Example Usage

### Basic Matcher

```terraform
resource "proxmox_virtual_environment_notification_matcher" "all" {
  name = "all-events"
}
```

### Matcher with Severity

```terraform
resource "proxmox_virtual_environment_notification_matcher" "warnings" {
  name             = "warning-and-above"
  match_severity   = ["warning", "error", "critical"]
  comment          = "Match warnings and higher"
}
```

### Matcher with Target

```terraform
resource "proxmox_virtual_environment_notification_matcher" "alerts" {
  name             = "critical-alerts"
  match_severity   = ["critical"]
  target           = ["smtp-alerts"]
  comment          = "Route critical alerts to SMTP"
}
```

### Matcher with Multiple Conditions

```terraform
resource "proxmox_virtual_environment_notification_matcher" "backup_errors" {
  name           = "backup-failures"
  match_field    = ["type=backup", "status=failed"]
  match_severity = ["error", "critical"]
  target         = ["sendmail-alerts", "gotify-alerts"]
  comment        = "Backup failure notifications"
}
```
